Research

Intelligent Transportation Systems & RL
Research Assistant – Intelligent Transportation Systems Lab
2024–2025 · Advisor: Dr. Monireh Abdoos, Shahid Beheshti University

Proposed SEMI-CTDE, a semi-centralized training, decentralized execution architecture for multi-agent deep RL traffic signal control that balances scalability with network-level coordination. Developed two realizations enabling coordinated behavior with minimal sensing in SUMO, improving network-wide traffic efficiency and peak-time performance over rule-based and decentralized baselines.

Bachelor Thesis – Multi-Agent DRL with Graph Neural Networks
2025–Present · Advisor: Dr. Monireh Abdoos

Thesis: “Multi-agent Deep Reinforcement Learning for Traffic Signal Control using Graph Neural Networks.” Benchmarking GNN-based agents against Double DQN baselines under identical SUMO setups, focusing on generalization across heterogeneous demand patterns and topology-aware state embeddings.

Selected Projects

Hands-On Work
  • Emotion Recognition from Text using DistilBERT
    GitHub Repository

    End-to-end pipeline for multi-class emotion classification on text. Fine-tuned DistilBERT in PyTorch, added an MLP classifier with label-specific thresholding and extensive hyperparameter search, and deployed a lightweight Streamlit app for interactive inference.

    NLP Transformers PyTorch Streamlit
  • LLM Armenia 2025 – Applied Research on Large Language Models
    ArmLLM GitHub Repository

    Implemented a Transformer encoder from scratch and trained it on the Cats vs. Dogs dataset to analyze learned representations and attention patterns. Built a vision–language model by connecting SigLIP image encoders to Qwen LLMs via multiple connector architectures, and explored inference-time reasoning strategies (majority vote, best-of-N) and adversarial attacks.

    LLMs Transformers Vision–Language Adversarial Robustness
  • Robotics: Indoor Mapping & Navigation
    GitHub Repository

    Developed aerial mapping with a Mavic 2 Pro and ground navigation with TurtleBot 3 in Webots. Implemented occupancy grid mapping, Monte Carlo Localization, and A* path planning to achieve fully autonomous indoor navigation.

    Robotics Webots Localization Path Planning
  • Connect Four AI Agent
    GitHub Repository

    Implemented a Minimax agent with alpha–beta pruning and a Q-learning baseline to compare search-based and reinforcement learning strategies in Connect Four.

    Reinforcement Learning Game AI Python
  • Custom Programming Language with LLVM Backend

    Designed a small C-like language with parsing, semantic analysis, and LLVM IR code generation, providing a complete front-end pipeline from source code to low-level representation.

    Compilers LLVM C++
